# 경기도청 교통상황 관리 시스템

## 개요
경기도청 교통정보과와 교통과를 위한 실시간 교통상황 관리 및 분석 시스템입니다.

## 주요 기능

### 1. 교통정보과 (관제/상황실) 기능
- **실시간 사고/돌발 인지 및 분류**: NLP 기반 자동 분류 및 우선순위 정렬
- **직관적 요약문/분류/상태 표시**: 신뢰도와 근거 하이라이트 제공
- **VMS 문구 자동 추천**: 한 번 클릭으로 전광판 송출, 수동 수정 가능
- **지도/영상/날씨 보조 정보**: 현장 대응 지원 (메시지 중심 UI)

### 2. 교통과 (정책/분석/도시계획) 기능
- **사고/돌발 데이터 축적 및 통계**: 도로·구간별 발생 현황
- **이력/분석 기능**: 정책 수립을 위한 데이터 기반 의사결정 지원
- **보고서/지표 자동화**: CSV, 차트, 보고서 내보내기

## 기술 스택
- **Backend**: Node.js, Express.js, Socket.IO
- **Frontend**: HTML5, CSS3, JavaScript, Bootstrap 5
- **Database**: SQLite3
- **NLP**: Natural.js (베이지안 분류기)
- **Charts**: Chart.js

## 사용법

### 메인 관제 화면
1. **상황 등록**: "상황 등록" 버튼으로 새로운 교통상황 입력
2. **상황 선택**: 목록에서 상황 클릭하여 상세 정보 확인
3. **VMS 발송**: 추천 메시지 사용 또는 직접 입력하여 전광판 발송
4. **상태 관리**: 해결/모니터링 버튼으로 상황 상태 변경

### 분석/정책 화면
1. **통계 조회**: 기간 및 카테고리별 필터링
2. **데이터 내보내기**: CSV 형태로 통계 데이터 다운로드
3. **시각화**: 차트를 통한 트렌드 분석

## 데이터베이스 구조

### traffic_incidents 테이블
- 교통상황 기본 정보 및 NLP 분석 결과 저장

### vms_templates 테이블
- VMS 메시지 템플릿 및 사용 통계

### statistics 테이블
- 일별/카테고리별/위치별 통계 데이터

## API 엔드포인트

### GET /api/incidents
활성 상태의 교통상황 목록 조회

### POST /api/incidents
새로운 교통상황 등록 (NLP 분석 포함)

### PUT /api/incidents/:id/status
교통상황 상태 업데이트

### GET /api/statistics
통계 데이터 조회 (기간/카테고리 필터링 지원)

### POST /api/vms/send
VMS 메시지 발송
